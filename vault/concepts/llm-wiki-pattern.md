# LLM Wiki Pattern

The LLM Wiki pattern stores accumulated knowledge as maintained Markdown pages instead of rediscovering facts from scratch on every query.

## Why it matters
- Supports compounding knowledge
- Makes synthesis persistent
- Encourages explicit cross-references
- Keeps the knowledge base inspectable by humans and tools
