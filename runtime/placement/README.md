# Runtime Placement

`runtime/placement/` owns topic, episode, node, and community placement helpers.

## Responsibilities

- place accepted memory into navigable topic and episode structures
- support Map Maker-style topology without owning semantic truth
- file topic pages and community bridges from already-bounded artifacts
- keep placement separate from capture, evaluation, and delivery

Placement may organize memory. It must not create canonical claims from raw
provider text on its own.
